# 更新日志

本项目采用 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/) 结构与语义化版本。

## [Unreleased]

### 新增

- 新增原生 macOS 13+ SwiftUI + AppKit 版本，支持菜单栏、22px/4px 边缘轨、辅助功能窗口跟踪和 Universal Intel/Apple Silicon 构建。
- 新增可 Fork 的品牌、主题、网站、目标应用配置，以及无 Secret 的 macOS GitHub Actions 构建链路。
- 应用 EXE、系统托盘和 NSIS 安装/卸载界面统一使用领哥 LOGO 多尺寸图标。
- 托盘全部可点击菜单项增加端到端命令分发测试，覆盖刷新、设置、更新、日志、排查与退出。
- 托盘新增“领哥个人网站”，使用系统默认浏览器打开固定 HTTPS 地址。

### 修复

- 详情卡改为点击 22px 轨才展开，并在移出、失焦、移动或进入 4px 紧凑模式时可靠收回，避免长期遮挡 Codex 顶部功能。
- 透明轨道显式拒绝 `WM_MOUSEACTIVATE`，点击详情时不再抢走 Codex 前景状态并导致弹层瞬间关闭。
- 主题画刷改用动态资源，托盘切换自动、深色或浅色后现有轨道与详情会立即更新。
- 额度轨通过 Win32 owner 关系附着到 Codex，最大化/全屏时 4px 轨不再被 Codex 窗口遮住，同时不会全局压住其他应用。

## [0.1.0-rc.1] - 2026-07-11

### 新增

- Windows x64 外侧贴边额度轨，支持 22px 双轨和 4px 紧凑模式。
- 官方 Codex App Server 额度读取、缓存、退避与自动恢复。
- 可用额度渐变、52% 失焦透明度、托盘菜单、可选开机自启、主题和减少动画。
- 睡眠/网络/Explorer/DPI 恢复、零遥测日志和手动 GitHub 更新检查。

### 已知限制

- 首发仅支持 Windows x64。
- 未配置发布证书时，安装器会显示“未知发布者”。
- WPF 版本在当前 QA 机器上的空闲工作集高于 80 MB，详见发布检查表。
- 当前候选版尚未配置 Windows 代码签名证书，安装器会显示“未知发布者”。
