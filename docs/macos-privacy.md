# macOS 隐私说明

macOS 版默认零遥测，不发送崩溃、行为或使用统计。

应用读取：

- 官方 Codex App Server 返回的账户是否可用与额度窗口；
- Codex 窗口的位置、大小、最小化、全屏和焦点状态；
- 本应用自己的版本化设置。

应用不读取：

- 屏幕像素或截图；
- 聊天、任务、项目文件或终端内容；
- 剪贴板；
- 邮箱、账号 ID 或访问令牌；
- 通讯录、相册、麦克风、摄像头或位置。

App Server 只通过子进程标准输入输出通信，不监听 TCP 端口。普通日志只记录时间、级别和固定事件名，不记录原始协议响应或远端错误文案。

辅助功能权限由 macOS 管理，可随时在“系统设置 → 隐私与安全性 → 辅助功能”撤销。撤销后边缘轨隐藏，菜单栏仍可用于退出和打开设置。
