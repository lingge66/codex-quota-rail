# 领哥个人网站托盘入口设计

## 目标

在 Windows 托盘右键菜单中增加“领哥个人网站”，让用户能从额度显示工具直接访问领哥主页。

## 交互

- 菜单项位于“故障排查”之后、“退出”之前。
- 点击后使用系统默认浏览器打开 `https://lingge66.pages.dev/`。
- 地址固定在应用代码中，不接受命令行或用户输入，避免任意协议与任意地址启动风险。
- 不弹出额外确认框，不影响额度条和详情窗口状态。

## 验证

- 托盘菜单测试锁定中文文案和相对位置。
- 命令测试覆盖从菜单 ID 到主机动作的完整分发链路。
- 桌面动作测试通过注入的启动回调校验固定 HTTPS 地址，不在自动化测试中真的打开浏览器。
- 发布前运行完整测试、Release 构建，并在真实 Windows 托盘菜单中做视觉核验。
