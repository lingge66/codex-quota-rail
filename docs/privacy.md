# 隐私说明

Codex Quota Rail 默认零遥测，不含分析 SDK、广告 SDK、崩溃上传或后台更新检查。

## 会处理的数据

- 官方 Codex App Server 返回的额度百分比、窗口时长和重置时间，仅在本机内存中用于显示。
- 本地设置：主题、减少动画、跟随暂停和开机自启偏好。
- 本地诊断日志：时间、级别、事件名和脱敏消息。

## 不会处理的数据

不会读取、记录或上传账号令牌、邮箱、账号 ID、聊天内容、项目文件、项目路径、剪贴板、键盘输入或 Codex Usage 网页内容。

## 网络访问

额度读取不访问互联网接口，而是连接本机 Codex App Server。只有用户从托盘点击“检查更新”时，程序才会请求 GitHub Releases API；发现新版本后也必须由用户确认才会打开浏览器。

## 本地文件

设置与日志位于 `%LOCALAPPDATA%\CodexQuotaRail`。卸载默认保留这些文件；用户可在卸载器中选择同时删除。删除该目录即可清除全部本地数据。
