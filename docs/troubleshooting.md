# 故障排查

## 没有显示边缘轨

1. 确认 Codex 桌面应用已启动并已登录。
2. 在托盘菜单中确认没有启用“暂停窗口跟随”。
3. 点击“立即刷新”，然后最小化并恢复 Codex。
4. 确认 Codex 版本支持本地 App Server；如提示不支持，请先更新 Codex。

## 显示“需要登录”或“不可用”

本程序不会收集登录信息。请直接在 Codex 中完成登录，再从托盘立即刷新。缺失或异常数据不会被当作 100%。

## 位置不正确

移动 Codex 到当前显示器后恢复一次窗口。跨屏或调整缩放比例时，程序会在 `WM_DPICHANGED` 后重新计算一次。如果仍不正确，请记录 Windows 缩放比例、显示器排列和 Codex 是否最大化。

## CPU 或内存异常

在应用稳定 15 秒后运行：

```powershell
.\scripts\manual-qa.ps1 -DurationSeconds 60
```

同时从托盘打开日志目录。不要公开包含个人目录名的原始日志；提交 Issue 前先检查并脱敏。

## 重置

退出程序，备份后删除 `%LOCALAPPDATA%\CodexQuotaRail`。再次启动会恢复默认设置。开机自启可随时从托盘关闭。
